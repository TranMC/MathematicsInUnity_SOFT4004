# Mathematics in Unity

Một dự án Unity đơn giản về toán học, bao gồm các script và scene để học tập và thử nghiệm các khái niệm toán học trong môi trường game engine.

## Nội dung

- **WorldToScreen.cs**: Script chuyển đổi tọa độ World sang tọa độ Screen trong Unity
- **SampleScene.unity**: Scene mẫu để demo các tính năng

## Yêu cầu

- Unity 2022 hoặc cao hơn
- .NET Framework tương thích

## Cách sử dụng

1. Mở project trong Unity Editor
2. Mở SampleScene để xem demo
3. Kiểm tra và sử dụng các script trong Assets/Scenes/
